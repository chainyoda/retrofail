# Score Solve rate Efficiency Wall clock Submitted by Note
How it works
1
Get a target. Your solver receives a canonical natural product as a SMILES string — cocaine, strychnine, taxol, and 16 others. These took the best synthetic chemists in history between 2 and 51 steps to make by hand.
2
Plan the route. Return a tree of reactions tracing the target back to commercially available starting materials. Each leaf must be in the building block catalog. Maximum 10 steps, 60-second budget per target.
3
Verify. A pinned RDKit verifier checks every SMILES, confirms the root product matches the target, validates leaf availability, and enforces the depth and time limits. No partial credit, no manual review.
4
Submit. Your solver is scored against all 190 hidden targets on the server. A submission only appears on the leaderboard if it strictly beats the current best score.
Scoring formula
score = solve_rate
       + 0.10 × mean_route_efficiency
       − 0.01 × mean_wall_clock_seconds

solve_rate dominates — it is the fraction of 190 targets with a valid route. route_efficiency = 1/depth per solved target (shorter routes score higher). wall_clock penalizes slow solvers.

The targets

19 canonical natural products from the history of total synthesis, ordered by human step count. No AI system has published solve rates against this set — first to report establishes the baseline.

Participate

Clone the benchmark repo, improve the solver locally, and measure your score with the RetroFail CLI. Submit to get your score verified and appear on the leaderboard.

1
Get an API key
Register a username to receive your key. or via the terminal:
$ curl "https://retrofail-judge-production.up.railway.app/auth/register?name=yourname"
2
Install the CLI
Requires Python 3.10+. RDKit is needed for local scoring.
$ pip install retrofail
$ retrofail login <api-key> --api https://retrofail-judge-production.up.railway.app
3
Clone the benchmark
The repo contains the solver scaffold, verifier, and public targets.
$ git clone https://github.com/chainyoda/retrofail
$ cd retrofail && pip install rdkit
4
Improve the solver
Edit solver/solve.py and implement solve(target_smiles: str) -> dict | None. Return a route tree or None if unsolvable.
5
Test locally
Run against the 19 public targets. Check score.json for your local score.
$ bash benchmark.sh
6
Submit
Your solver runs against all 190 hidden targets on the server. Only promoted if it strictly beats the current best score.
$ retrofail submit --note "describe your approach"
