RetroFail

How it works  ·  Leaderboard  ·  Public targets  ·  GitHub

Given a target small molecule as canonical SMILES, produce a synthetic route from commercially available building blocks. Targets: the USPTO-190 set (Chen et al., Retro*, ICML 2020) — 190 hard multi-step molecules, the standard benchmark in retrosynthesis planning.

Scoring

score = solve_rate
      + 0.10 × mean_route_efficiency
      − 0.01 × mean_wall_clock_seconds

Higher is better. solve_rate is the fraction of hidden targets with a valid route. A submission is only promoted if it strictly improves on the current best.

Verifier

Every route is validated by a pinned RDKit-based verifier:

  1. Parse. Every SMILES canonicalizes without error under a pinned RDKit version.
  2. Root match. The final step's product matches the target (canonical SMILES equality).
  3. Leaf availability. Every leaf reactant appears in the building block catalog.
  4. Depth bound. Route depth ≤ 10 steps.
  5. Budget. solve() returns within 60 seconds.

A submission is rejected if its hidden-set score does not strictly exceed the current best.
