Frontier Benchmark · Drug Discovery

RetroFail

Can AI plan better synthetic routes for the hardest molecules in drug discovery?

The Problem

Given a target molecule, find a step-by-step synthesis from commercially available building blocks. Existing tools fail badly on PROTAC linkers and macrocycles — two chemotypes central to next-generation drugs.

The Opportunity

PROTAC programs burn 3–6 months and $500K–$2M per campaign on linker synthesis planning. A 10-point solve-rate improvement keeps ~20 drug programs alive per campaign that would otherwise be dropped as "unsynthesizable."

Why Now

No open, executable benchmark exists for this chemotype. New ML retrosynthesis models (LocalRetro, Graph2Edits) are ready to be tested. A public leaderboard gives the field a single number to compete on.

How it works

A machine-verifiable benchmark with a single scalar score and a strict promotion rule.

score =   solve_rate  + 0.10 × efficiency  − 0.01 × wall_clock

Compete in 4 steps
1

Clone

Get 10 public target SMILES + a pinned RDKit verifier. Test locally against the same scoring logic the server uses.

2

Implement solve(smiles) → Route

Any approach: SMARTS templates, MCTS, learned one-step model, LLM. Only the route JSON is evaluated.

3

Submit

Server runs your solver against 100 hidden targets. Routes are validated: correct SMILES, purchasable leaves, depth ≤ 10, within 60s.

4

Promote (or don't)

Your submission only hits the leaderboard if it strictly beats the current best. No gaming the public set.
